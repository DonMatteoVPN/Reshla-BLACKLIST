# Reshla-BLACKLIST

Automated Community Justice System "Judge Dredd" style.

## Features
- **Voting System**: Users report bad actors, community votes.
- **Auto-Ban**: If >30 votes in 1h, user is banned.
- **Moderation**: If generic report, goes to moderation queue.
- **GitHub as Database**: All data stored in Issues and Git.

## Setup
1. Clone repo
2. `npm install`
3. `npm run dev`

## Deployment
Deployed on GitHub Pages via Actions.
